The `<VirtualizedList />` component. This component is used virtualize a list being passed in as a
child. This component accepts **setListData** state function that updates the list data, an
**onScroll** function to envoke as the user scrolls through the list, and a wide range of props that
get passed to the virtualizer. To see a complete list of props, refer to
[Tanstack Virtualizer API Docs](https://tanstack.com/virtual/latest/docs/api/virtualizer)
